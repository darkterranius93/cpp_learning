 - char - символьный
 - wchar_t - символьный двубайтовый

# Ключевое слово `typedef`
Ключевое слово `typedef` позволяет создать новое имя для существующего типа данных:
```cpp
typedef тип новое_имя;
```

# Ключевое слово `sizeof`
Позволяет узнать размер (в байтах) одного из типов данных или значения. Оператор получет размер во время компиляции.

```cpp
sizeof (type)
sizeof value
```

# Динамическое распределение памяти с использованием операторов `new` и `delete`
Данные операторы выполняют функции по выделению и освобождению памяти
```cpp
переменная_указатель = new тип_переменной;
delete переменная_указатель;
```
**Пример**
```cpp
int *p;
p = new int; // Выделяем память для int-значения
*p = 20; // Помещаем в эту область памяти значение 20
std::cout << *p << std::endl; // Убеждаемся (путём вывода на экран) в работоспособности этого кода
delete p; // Освобождаем память
```

**Инициализация динамически выделенной памяти**
```cpp
int *j;
j = new int(99); // Инициализируем память числом 99
std::cout << *j << std::endl; // На экран выводится 99
delete j; // Освобождаем память
```

**Выделение памяти для массивов**
```cpp
double *array;
int i;
array = new double [10]; // Выделяем память для 10-элементного массива
// Заполняем массив значениями от 100 до 109
for (i = 0; i < 10; i++) {
    array[i] = 100.0 + i;
}
// Отображаем содержимое массива
for (i = 0; i < 10; i++) {
    std::cout << array[i] << " ";
}
std::cout << std::endl;
delete [] array; // Удаляем весь массив
```
